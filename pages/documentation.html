Getting Started with SafiTrack

Welcome to the SafiTrack documentation. This guide will help you set up your organization and start tracking field activities in under 10 minutes.

💡
New to SafiTrack? Check out our onboarding tips for high-performing teams.

Quick Install

For mobile users, SafiTrack is a Progressive Web App (PWA). You can install it directly from your browser without visiting an app store.

// To integrate with our JS SDK
npm install @safitrack/sdk-core

// Initialize with your API key
const safi = new SafiTrack({
  apiKey: 'st_live_...',
  region: 'africa-west-1'
});

Authentication

All API requests must include a Bearer token in the Authorization header. You can generate tokens in your Organization Settings.

Parameter Type Description
api_key String Your unique organization identifier
env Enum Either production or sandbox
⚠️
Important: Never expose your secret API keys in client-side code (frontend JavaScript).

Core Concepts

SafiTrack is built around three primary entities that allow for maximum flexibility in tracking field workflows.

  • Companies: Professional entities that your sales reps visit or service.
  • People: Individual contacts within those companies.
  • Visits: Verified geolocation check-ins performed by your field team.
