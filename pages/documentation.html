Quick Start

Get your team set up in under 10 minutes by following these steps:

  • Create a SafiTrack account at /crm/
  • Import contacts via CSV or connect a Google Sheet
  • Add your first company and assign reps
  • Log your first sales visit or create a route plan

Authentication

SafiTrack uses Supabase Auth under the hood. All API calls require a Bearer token obtained after sign-in.

curl -X POST https://ndrkncirkekpqjjkasiy.supabase.co/auth/v1/token \
  -H "apikey: YOUR_ANON_KEY" \
  -H "Content-Type: application/json" \
  -d '{"email":"you@company.com","password":"yourpassword","grant_type":"password"}'

Contacts & Companies

Every person and business in your pipeline lives in SafiTrack's contact graph. Push contacts via the REST API or import them in bulk using CSV files up to 50,000 rows.

// Fetch all companies
const { data, error } = await supabase
  .from('companies')
  .select('*')
  .order('created_at', { ascending: false });

Sales Visits

Each visit record captures GPS coordinates, duration, notes, outcomes and attachments. Visits power the Intelligence Hub's AI summaries and the weekly performance digest.

Webhooks

Subscribe to real-time events using webhook endpoints. SafiTrack sends a POST request to your URL whenever a trigger fires.

// Example webhook payload
{
  "event": "visit.completed",
  "timestamp": "2026-01-28T09:41:22Z",
  "data": {
    "visit_id": "abc123",
    "rep_id": "rep_456",
    "company_id": "co_789",
    "outcome": "demo_booked"
  }
}
